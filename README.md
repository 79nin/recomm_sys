# recomm_sys — Гибридная рекомендательная система с MLOps и A/B тестированием

## Описание проекта
Проект представляет собой end-to-end решение для построения рекомендательной системы:
- Гибридный подход (коллаборативная фильтрация + контентные признаки)
- Сервинг модели через FastAPI
- Векторный поиск (FAISS / Qdrant)
- Оркестрация пайплайнов (Prefect / Airflow)
- Мониторинг дрейфа данных (Evidently AI)
- A/B тестирование с оценкой статистической значимости

## Структура репозитория
- `notebooks/` — Jupyter ноутбуки для экспериментов
- `src/` — исходный код проекта (пакет Python)
- `data/` — директория для данных (игнорируется Git)
- `config/` — конфигурационные файлы
- `requirements.txt` — зависимости Python

## Начало работы
1. Клонируйте репозиторий
2. Создайте виртуальное окружение: \`python -m venv venv\`
3. Активируйте его и установите зависимости: \`pip install -r requirements.txt\`
4. Скачайте датасет (например, Retailrocket) и поместите в \`data/raw/\`
5. Запустите ноутбуки из папки \`notebooks/\` для анализа и прототипирования
